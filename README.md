Simple-Base64
=============

Simple-Base64 is a small Unix tool for encoding and decoding Base64 strings

Usage
-----
```sh
$ b64 SXQgd29ya3Mh
> It works!

$ b64 "It works!"
> SXQgd29ya3Mh
```

Installation
------------
```sh
$ curl https://raw.github.com/ShaneCurran/Simple-Base64/master/install.sh | sh
```
